User settings

UserPanel provides simple yet powerful interface for users to manage their user settings. it allows users to change their profile info and change user password.

User profile settings

UserPanel's user profile settings page can be used to update users profile page info.

User account settings

UserPanel's user account settings page can be used to change users user password.
