User profiles

Both UserPanel administrators and users have access to their own user profile. it can be easily access by click on small user thumbnail at the top right corner of UserPanel.

User info

Each and every user's user profile has some basic info about the user.

Profile picture

UserPanel comes with built-in Gravatar support. so users can easily change their Avatar on Gravatar.
