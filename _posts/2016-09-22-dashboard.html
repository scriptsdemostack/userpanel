Administrator dashboard

UserPanel administrators are able to view some cool stats on UserPanel dashboard. They can see how many new users are registered on today, yesterday, last 7 days, last 28 days, total registered user count, online user count and monthly basis new user registration chart.

Also UserPanel administrators can see how many registered users are in different user statuses. not just only user counts in different user statuses, also giving easy access to those user status groups.

Also UserPanel administrators can see recently registered users with some basic info. administrators can view those user's profiles by clicking on them.
