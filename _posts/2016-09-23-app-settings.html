App settings

UserPanel provides simple yet powerful interface for manage global UserPanel app settings. it allows administrators to easily enable/disable or change UserPanel global settings.

General app settings

UserPanel's general app settings page can be used to update UserPanel application name and email sending email address.

User app settings

UserPanel's user app settings page can be used to enable/disable system features or change system variables on UserPanel application.
